# bbzw-m347

Dieses Repository enthält Übungen und Projekte aus dem Modul **M347 – Dienst mit Container anwenden**.

## Struktur

```
DL03/   Python-Skripte für Docker SDK Übungen
DL04/   Containerisierte API-Anwendung
```

Jeder DL-Ordner enthält eigenständige Aufgaben mit eigenem Code und ggf. eigener virtueller Python-Umgebung.

## Voraussetzungen

* Python 3.12
* Docker
* Git

## Virtuelle Umgebung

Eine virtuelle Umgebung kann pro DL oder Projekt erstellt werden:

```
python -m venv .venv
source .venv/bin/activate
```

## Abhängigkeiten installieren

```
pip install -r requirements.txt
```

## Docker

Docker wird in den späteren Übungen verwendet, um Anwendungen in Containern auszuführen.
